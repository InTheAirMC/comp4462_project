# 🎬 Oscar Bait Movies
## An Interactive Scrollytelling Visualization

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This interactive visualization explores the relationship between Oscar-nominated movies and the concept of **"Oscar Bait"** — films that seem engineered to win awards rather than to delight audiences.

Through a narrative-driven, scroll-based interface, the project analyzes what makes a movie Oscar Bait by looking at:

- **Nominated categories** – Are technical awards a strong indicator?
- **Genres** – Which genres produce the most "bait"?
- **Actors involved** – Which stars frequently appear in these films?
